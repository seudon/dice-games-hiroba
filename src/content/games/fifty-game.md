# 50ゲーム

## ゲーム概要

サイコロを2個振って出た目の合計を累積し、**合計が50ぴったり**になるまで挑戦するゲームです。
ただし、50を超えたらバースト（失敗）！いつ止めるかの判断が勝負の分かれ目です。

## 基本ルール

1. **サイコロを振る**: 2個のサイコロを振ります
2. **合計を加算**: 出た目の合計を現在のスコアに加算します
3. **継続か停止か判断**:
   - **続ける**: もう一度サイコロを振る（リスク大）
   - **止める**: 現在のスコアで終了（安全策）
4. **勝利条件**: 合計がちょうど50になったら勝ち！
5. **バースト**: 50を超えたら失敗、最初からやり直し

## 戦略のポイント

### 🎯 いつ止めるべきか？

- **35-40点**: まだ余裕があります。続行推奨
- **41-45点**: 慎重に！バーストのリスクが高まります
- **46-48点**: 危険ゾーン！2が出ればバースト
- **49点**: 究極の選択！1が出れば勝ち（確率1/6）

### 📊 確率を考える

サイコロ2個の出目の合計確率:
- **2**: 1/36 (約2.8%)
- **3**: 2/36 (約5.6%)
- **7**: 6/36 (約16.7%) - 最も出やすい
- **12**: 1/36 (約2.8%)

現在のスコアが46以上になったら、バーストのリスクを計算しましょう！

## 遊び方のコツ

1. **序盤（0-30点）**: 積極的に振って点数を稼ぐ
2. **中盤（31-40点）**: バランスを考えて慎重に
3. **終盤（41-48点）**: リスク計算が重要
4. **49点**: 勝負をかけるか、次回に期待するか

## 実際のサイコロで遊ぶ場合

1. サイコロ2個を用意
2. 紙とペンで累計スコアを記録
3. 電卓があると便利
4. 複数人で遊ぶ場合は、誰が最少ターンで50に到達するかを競う

## バリエーション

- **チャレンジモード**: 10ターン以内に50到達を目指す
- **対戦モード**: 2人以上で、先に50に到達した人が勝ち
- **ハードモード**: サイコロ1個で挑戦（より時間がかかる）
- **目標変更**: 50ではなく100を目指す

## ゲームの魅力

- **シンプルだけど奥深い**: ルールは簡単、でも戦略は深い
- **運と判断力**: 単なる運ゲーではなく、止め時の判断が重要
- **短時間で遊べる**: 1ゲーム数分で終わる
- **教育的**: 足し算と確率の練習になる
