# サイコロポーカー

## ゲーム概要

5個のサイコロを使って、ポーカーのような「役」を作るゲームです。
各ターン3回まで振り直しができ、どのサイコロをキープするかが戦略のカギ！
全13ラウンドで各役を1回ずつ使い、合計得点を競います。

## 基本ルール

### ゲームの流れ

1. **サイコロを振る**: 5個のサイコロを一度に振ります
2. **キープを選択**: 残したいサイコロをクリックしてキープします
3. **振り直し**: キープしなかったサイコロを振り直します（最大2回まで）
4. **役を確定**: 取りたい役を選んで得点を記録します
5. **次のラウンド**: これを13ラウンド繰り返します

### 振り直しのルール

- 各ラウンド最大**3回**まで振れます（初回+振り直し2回）
- 振り直し時に任意のサイコロをキープ可能
- キープしたサイコロは次回振られません
- 振り直しはスキップして早めに役を確定することも可能

### 役の選択

- 各役は1ゲーム中に**1回だけ**使用できます
- 一度記録した役は変更できません
- 全13種類の役を使い切ったらゲーム終了

## 役一覧と得点

### 上段（数字の合計）

| 役 | 条件 | 得点 |
|---|---|---|
| **1の目** | 1の目の合計 | 1 × 個数 |
| **2の目** | 2の目の合計 | 2 × 個数 |
| **3の目** | 3の目の合計 | 3 × 個数 |
| **4の目** | 4の目の合計 | 4 × 個数 |
| **5の目** | 5の目の合計 | 5 × 個数 |
| **6の目** | 6の目の合計 | 6 × 個数 |

**ボーナス**: 上段合計が63点以上で **+35点**
（各数字を3個ずつ揃えると達成可能）

### 下段（特殊役）

| 役 | 条件 | 得点 |
|---|---|---|
| **スリーカード** | 同じ目が3個以上 | 出目の合計 |
| **フォーカード** | 同じ目が4個以上 | 出目の合計 |
| **フルハウス** | 3個+2個のペア | **25点** |
| **スモールストレート** | 4個連続の数字 | **30点** |
| **ラージストレート** | 5個連続の数字 | **40点** |
| **ヤッツィー** | 全て同じ目（5個） | **50点** |
| **チャンス** | 何でもOK | 出目の合計 |

### ストレートの例

- **スモールストレート**: 1-2-3-4, 2-3-4-5, 3-4-5-6 のいずれか
- **ラージストレート**: 1-2-3-4-5 または 2-3-4-5-6

## 戦略のコツ

### 上段のボーナスを狙う

上段合計63点以上で**+35点**のボーナスがもらえます。
4の目、5の目、6の目は積極的に狙いましょう。

**例**: 6の目を3個揃えれば18点（6×3）

### 振り直しのタイミング

- **1回目で良い出目が出たら**: 早めに確定するのもアリ
- **あと1個で役が揃う**: 振り直してチャレンジ
- **残り回数に注意**: 3回目は振り直せません

### チャンスの使い方

「チャンス」は出目の合計なので、大きい数字が多い時に使うのがお得。
例: 6-6-6-5-5 = 28点

### 捨て役を決める

全ての役で高得点は難しいので、捨てる役を決めるのも戦略。
例: 1の目や2の目に低い得点を入れて、他で挽回

## ヒント機能

ゲーム中に「おすすめの役」が表示されます。
迷った時は参考にしてみましょう！

## 統計記録

- **最高得点**: あなたのベストスコア
- **平均得点**: これまでの平均
- **ゲーム数**: プレイ回数
- **ヤッツィー達成回数**: 5個揃った回数

## 実際のサイコロで遊ぶ場合

このゲームは実際のサイコロでも楽しめます:

1. サイコロ5個と紙とペンを用意
2. スコアシートを作成（役一覧表）
3. 振り直し回数を正の字で記録
4. 友達や家族とスコアを競いましょう

## バリエーション

- **ハイスコアチャレンジ**: 300点超えを目指す
- **対戦モード**: 友達と交代で1ゲームずつプレイして得点を競う
- **制限時間**: 各ラウンド30秒以内など時間制限を設ける
